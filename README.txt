**********************************************************************

                        WELCOME TO GTKPP

**********************************************************************

OS targeted : Linux ( & currently only installed binaries)
gtk-version : gtk3.x/gtkmm3.x

Compiling C/C++ programs that uses GTK graphics libraries is not
an easy task given that  you have include the right headers and 
link to the right libraries.


GTKPP provides a bash wrapper on top of gcc/g++ to facilitate
easy compilation of C/C++ projects involving gtk/gtkmm

Installation instructions

Copy gtkpp to /usr/bin and do as root

   chmod +x gtkpp
